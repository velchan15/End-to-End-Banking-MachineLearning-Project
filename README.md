# End-to-End-Banking-MachineLearning-Project
A comprehensive Machine Learning project on banking data, integrating Unsupervised Clustering and Supervised Classification

Project Overview: Proyek ini bertujuan untuk mengelompokkan nasabah bank berdasarkan pola transaksi (Clustering) dan membangun model klasifikasi untuk memprediksi kelompok nasabah baru.

Tech Stack: Python, Scikit-Learn, PCA, K-Means, Decision Tree, Random Forest.

Key Features:

- EDA & Preprocessing: Handling outliers, feature scaling, and binning.

- Clustering: PCA dimensionality reduction & Elbow Method.

- Classification: Hyperparameter tuning menggunakan GridSearchCV.
